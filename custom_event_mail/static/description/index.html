PPTS: Custom Event Notification

In Odoo PPTS: Custom Event Notification module used to create and customize notification(Mail) as per customer and attendees and therapist needs in the Events.

Step 1:

In Events Module click on Configuration -> Notification -> Event Notification to create new event notification.

Step 2:

After Creating new notification record, at first Notification Active Button is used to Activate in every notification titles. If it is deactivate the notification cannot send to any customers or attendees.

Step 3:

There are sixteen type of Event notification, in every notification title there is a activate button. These sixteen events notification will send mail/notification to the specific attendees in each operation done in events records.

  • Active button - To enable mail/notification
  • Mailing list - To send mail to the selected users
  • Mail Template - To select mail template/format for every notification
  • Whatsapp Template - To select wahtsapp template for sending whatsapp notification

Step 4:

The Mailing List has been created from the Email Marketing -> Mailing List -> Create a new mailing list with mailing list contacts and events mail template and whatsapp template.For mailing list users the notification need to customize in maillist form itself,because that users have seperate rules for mail/notifications.The mailing list contacts is shown in the Recipients Smart button.

Step 5:

Notifications

Events Notifications Description
1.Event Creation The Event Creation notification will send notification to the mailing list contacts if selected, by default mail will not send to any user on creating.
2.Event Approval The Notification will send to the attendees on approval of events.
The default mail template is event attendees approval, and also have a option to change template.
If whatsapp template also selected, the notification will send through mail.
On Selecting the mailing list, the notification is also send to Mailing list contacts.
3.Event Cancel The Notification will send to the attendees on cancelling of events.
The default mail template is event attendees cancel with reason, and also have a option to change template.
If whatsapp template also selected, the notification will send through mail.
On Selecting the mailing list, the notification is also send to Mailing list contacts.
4.Event Reschedule The Notification will send to the attendees on rescheduling the event by the attendees or user for som emergency reasons.
If whatsapp template also selected, the notification will send through mail.
On Selecting the mailing list, the notification is also send to Mailing list contacts.
5.Room Allocation The Notification will send to the attendees on allocating room for the events.
In this, the mail will send to mailing list contacts,if Mailing list has been selected.
it will send mail through mail and whatsapp to the attendees.
6.Event Checklist The Notification will send to the attendees on creating checklist for the events.
In this, the mail will send to mailing list contacts,if Mailing list has been selected.
it will send mail through mail and whatsapp to the attendees.
7.Attendees Booking The Notification will send to the attendees on booking the events.
In this, the mail will send to mailing list contacts,if Mailing list has been selected.
it will send mail through mail and whatsapp to the attendees.
8.Attendees Booking Confirmation The Notification will send to the attendees on confirmation of booked events.
In this, the mail will send to mailing list contacts,if Mailing list has been selected.
it will send mail through mail and whatsapp to the attendees.
9.Attendees Waiting List The Notification will send to the attendees if the customer is in waiting list.
In this, the mail will send to mailing list contacts,if Mailing list has been selected.
it will send mail through mail and whatsapp to the attendees.
10.Attendees Booking Cancellation The Notification will send to the attendees if customer cancelled the event on that date.
In this, the mail will send to mailing list contacts,if Mailing list has been selected.
it will send mail through mail and whatsapp to the attendees.
11.Attendees Reminder mail + What's app before 2 hours The Notification will send to the attendees before 2 Hours need to attend.
it will send mail through mail and whatsapp to the customer and staff.
12.Attendees Reminder mail before 3 day for events The Notification will send to the attendees before 3 days for events.
it will send mail through mail and whatsapp to the customer and staff.
13.Attendees Reminder mail before 1 day The Notification will send to the attendees before 1 day events need to attend.
In this, the mail will send to mailing list contacts.
it will send mail through mail and whatsapp to the customer and staff.
14.Attendees after attended the events The Thanks Notification will send to the attendees after events attended.
In this, the mail will send to mailing list contacts.
it will send mail through mail and whatsapp to the customer and staff.
15.Attendees Payment Reminder This Notification will send to the attendees after completed to remain the payment.
In this, the mail will send to mailing list contacts.
it will send mail through mail and whatsapp to the customer and staff.
16.Invoice & Payment If the payment/invoice is paid it will send notification for payment completed status
In this, the mail will send to mailing list contacts.
it will send mail through mail and whatsapp to the customer and staff.
