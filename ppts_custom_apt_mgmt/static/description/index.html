Custom Appointments Management

Custom Appointments Management module is use to book appointment for a customers.

Step 1: Creating Employee

For Booking Appointment the first is to create a Therapist in an Employee module to select a therapist for an appointment. And need to create service details for a therapist. In a services details the therapist need to select services category, subcategory, the duration for the service provided, and price.

Step 2: Creating Availability

After creating a therapist, the therapist needs to create availability for services. In an availability module, create an availability by selecting which therapist need to be available and the service category that the therapist will provide in that availability and select which services are active or inactive and to set staff available or not and the venue will be picked up from the employee location.

Step 3:

After Venue, the user can select therapist availability start date and end date, which makes the user select many dates, and can user can select start time and end time for the selected dates. On clicking on the Set Dates button, the selected dates have been listed below, on that list, the user can edit the start time and end time for each line and the user can delete some specific date also. Finally, On clicking on Create Availability Button the Availablity have been created and State has been changed from Draft to Availability Created.

Step 4: Appointment List View

In an Appointment list view, the appointment is filtered by Upcoming Appointments. Appointment list view shows:

  • Appointment creation date
  • Status of an appointment(booked/confirmed/completed/cancelled)
  • Customer Name
  • Type of session that customer consumed
  • Service Category/Sub Category/Therapist that customer selected
  • Appointment Date
  • An Appointmnet start & end time customer select
  • Payment Status

Step 5: Creating Appointment

The First step is to select customer is existing or new.

If the Customer is existing select the customer name from the contacts, it will get the customer mail-ID and Mobile Number and History of a customer. The customer name can search by mail-ID and Mobile Number also.

If the Customer is new, Create a new contact by filling required field from Appointmnet form itself

Step 6:

After Selecting Customer, the Venue will get from customer location, Sales Rep from the current user, Creation date is a current date Lead Source: from where the customer get know about illuminations, and Mode of appointment is how the appoinment booked.

Step 7:

In a Service Details there are two type of session available, Single Session and Package Session.

Single Session

In Single Session, Select the appointment date that customer needs, Therapist available, Service Category, Sub Category, Appointment Platform as On-site/Online.THe service category and sub category will display according to therapist.

After Selecting the therapist, the booking info tab will enable with Duration and time slot. The Duration and time slot will display according to therapist.And customer can select which type of room he/she need. "Do you want to edit time" checkbox is use to enable editing of time.

The Unclaimed package sessions of a customer on that day is shown in the below Unclaimed package sessions tab with the package details, If the customer needs, he/she can claimed it and can confirm it in that tab itself.

Package Session

If the customer select Package Session, first need to select therapist, service category, sub category, duration of the therapy the customer needs, and at last need to select which packages that the customer needs(packages of 3/5/10) that available by the therapist.

On clicking "Add packages" button the packages has been listed in the Packages infomation tab

Void button in the package is use to cancel the package.If one of the appointment in a packages in confirmed, the user cannot cancel the package.

The booked packages is claim by single session for the packages.

Click on move to POS for payment. The view cart smart button shows the packages booked.

In the POS Screen Select the order that customer booked.

In the POS Screen Confirm the packages booked and of customer want another package cant select it in cart, adn click on Payment button to make payment.

In Payment Tab Select the Payment menthod as Cash/Net Banking/G-pay/Bank and click on Validate to completed the payment

After Payment done it will generate the invoice for that payment and can have a option to send invoice in mail.

After claimed and confirm the package in single session, the appointment date, time slot will confirmed and status changed to availed.

On Customer arrived, Clickon Arrived button to change appointment status as Arrived and payment status will change to paid for package.

After the appointment completed click on Completed button to end the appointment and status will move to Completed
