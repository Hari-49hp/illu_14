Manage phone campaigns and dial contacts.

Connect calls to your agents or playback announcements.

Features

  • Three types of campaigns available - Operator campaigns, Sound file playback and Google TTS.
  • Import your contacts from Odoo contacts manually or automatically using pre-defined rules.
  • Run your campaigns manually or schedule them - can be run periodically or between two dates.
  • Campaign logs and call history
  • Tight Odoo integration
  • All Asterisk distros are supported.
